نبذة عن المشروع

هذا المشروع عبارة عن صفحة ويب شخصية قمت بإنشائها لعرض معلوماتي الشخصية وتخصصي الأكاديمي والتدريب الذي أقوم به حاليًا.

خطوات تنفيذ المشروع
بدأت باستخدام قالب HTML من موقع W3Schools.
قمت بتعديل الكود ليتناسب مع بياناتي الشخصية.
أضفت صورة إلى الصفحة بالاستعانة بالأكواد الموجودة في موقع W3Schools.
كتبت وعدلت الكود باستخدام برنامج المفكرة (Notepad).
حفظت جميع ملفات المشروع، بما في ذلك ملف HTML والصورة المستخدمة.
أنشأت استضافة مجانية على موقع InfinityFree.
رفعت ملفات الموقع والصورة من خلال File Manager في InfinityFree.
بعد رفع الملفات تم نشر الصفحة وأصبحت تعمل بنجاح على الإنترنت.
